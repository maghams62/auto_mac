# Scripts Directory

This directory contains utility scripts, debug scripts, and one-off development tools.

## Contents

### Debug Scripts
- `debug_stock_page.py` - Debug script for stock page automation
- `demo_maps_llm.py` - Demo script for maps LLM functionality

### Verification Scripts
- `verify_agent_distinction.py` - Verify agent distinctions in the system
- `verify_maps.py` - Verify maps agent functionality

## Usage

These scripts are typically run directly for debugging or development purposes:

```bash
python scripts/debug_stock_page.py
python scripts/verify_maps.py
```

## Note

These scripts are development/debugging tools and may not be maintained as rigorously as the main codebase. Use at your own discretion.
